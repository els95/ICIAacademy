반복문(for 문)
for문의 기본 구조
 - for(초기식; 조건식; 증감식){             <---- 증감식에는 세미콜론 쓰지 않음
                                                
    실행문;

 } 

 순서 -> 초기식, 조건식, 실행문, 증강식
 - 초기식은 한번만 선언하여 조건식으로 넘김
 - 실행문으로 실행시켜 값 출력 후 다시 증감식으로 넘김
 - 조건에 맞을 때까지 반복


Continue문
1) 반복문과 함께 사용
2) 반복문 안에서 Continue문을 만나면 이후 문장은 실행하지 않고, for문에 증감식으로 이동한다.

중첩for문
- for문 안에 for문 사용
for(초기식; 조건식; 증감식){          <--------- 외부for문

    for(초기식; 조건식; 증감식){            <-------- 내부for문  

     실행문;

    } 
}
