VariableScope(변수영역)
전역변수 (신용카드)
 - script 어디에서나 사용 가능한 변수

지역변수 (이음카드)
 - 해당 함수에서만 사용가능한 변수로 해당 함수의 밖이나 다른 함수에서 사용할 수 없다.


객체(Object) 
- 다른것과 식별이 가능한 것
- 자신만의 고유한 특성과 행동을 지니며 다른 객체들과 상호작용을 한다.

- 객체를 만드는 방법, object를 선언하는 방법
     (1) var object1 = {};
     (2) var object2 = new object();
        - key : value 가 한 쌍

- innerHTML vs innerTEXT
 :innerHTML은 html 태그를 적용하지만 innerTEXT 모든 문자 그대로 반영
