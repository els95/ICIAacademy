함수(Function) : 특정 동작(기능)을 수행하는 일부 코드의 집합. 함수를 실행 -> 함수를 호출한다.
    - 함수의 기본 구조
         function 함수이름(){
            실행하고자 하는 내용;}

    - 함수를 호출하는 방법
        함수 이름();
    - 함수이름 뒤에 괄호안에 들어가는 데이터는 매개변수(파라미터)라고 한다.
      매개변수는 함수 안에서만 사용이 가능하다.

    -minus함수(빼기), multi(곱하기), divide(나누기)

Switch 조건문 : 단독 사용 불가. case와 같이 사용해야함. 변수는 제한 없음
    ex) switch(조건변수){
        case 변수1 : 
            조건변수가 변수1일때 실행;
            break;

        case 변수2 : 
            조건변수가 변수1일때 실행;
            break;

        case 변수n : 
            조건변수가 변수1일때 실행;
            break;    

        default :
            break;
    }

    - break문 : switch 문을 멈추게 하는 역활, 생략할시 모든 입력 문자들을 출력
